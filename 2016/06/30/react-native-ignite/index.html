react-native-ignite安装

  • react-native
星期六, 七月 2日 2016

在mac os系统中通过npm安装react-native-ignite时，会出现一个异常的错误，导致无法安装此react-native的启动包。通过对错误信息进行分析，发现问题存在于npm i -g generator-react-native-ignite这个命令中，经过一番探索，目前的解决方法有两个。

1.修改为ROOT用户

可以注销当前用户，使用ROOT用户进行登录，使用npm install -g react-native-ignite 即可。

2.先安装generator-react-native-ignite

  • 执行sudo npm install -g generator-react-native-ignite；
  • 执行sudo npm install -g react-native-ignite即可

3.使用react-native-ignite（ignite new MyProject）过程总存在的错误

  • 提示需要安装yo，且正常程序执行报错：使用sudo npm i -g yo
  • 提示确实rnpm：使用sudo npm i -g rnpm
  • 提示/usr/local/lib/node_modules/generator-react-native-ignite/app/templates：

sudo mkdir /usr/local/lib/node_modules/generator-react-native-ignite/app/templates

  • 提示Trying to copy from a source that does not exist: /usr/local/lib/node_modules/generator-react-native-ignite/a pp/templates/ignite-base/README.md.template

cd /usr/local/lib/node_modules/generator-react-native-ignite/a
pp/templates/
git clone https://github.com/infinitered/ignite/tree/master/ignite-base
